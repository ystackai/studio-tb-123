Pulse

live instrument

6-track step sequencer · polyphonic synth engine · drum machine · real-time Web Audio · MIDI keyboard support

▸ Pulse — Quick Guide

Sequencer

  • Space Play / Stop
  • 1-6 Select track
  • Q-P Toggle steps on selected track
  • A Clear pattern
  • R Randomize pattern
  • Ctrl+Click Toggle step
  • Click cells to toggle steps

Sound Engine

  • Kick: sine pitch-drop drum
  • Snare: noise burst with HP filter
  • Hat: noise with HP filter
  • Synth1/2: polyphonic oscillator with envelope
  • Bass: sawtooth with LP filter
  • Waveform selector for synth voices

Live MIDI Play

  • Web MIDI keyboard support
  • Notes C2-B3 (MIDI 36-59) mapped to synth
  • Velocity-sensitive envelope
  • Auto-connecs MIDI on first note

Effects

  • Lowpass filter with cutoff + resonance
  • Delay with mix control
  • Overdrive distortion on master
  • Swing control for groove

Track Controls

  • Per-track mute/solo
  • Per-track volume slider
  • 4 preset patterns + save to URL
