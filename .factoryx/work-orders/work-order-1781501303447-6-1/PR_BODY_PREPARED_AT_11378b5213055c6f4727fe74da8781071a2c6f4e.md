## Implemented (polish_until_deadline)

- Ambitious, first-screen-playable 2D arcade: 3-lane vertical acid circuit racer with independent polarity (G/P/B color) matching for "breaker" gates.
- Core verbs: lane switch (left/right or side taps), polarity cycle (center tap/SPACE), dodge glitches, collect pulses, dual-match (lane+color) to shatter gates for score+combo.
- Beat-synced: gates crossed near visual beat peak award phase bonus (1.5x + pip); world grid + elements react to polarity + beat.
- Escalating patterns from LVL2 (color-hold sequences or lane-locked + glitch barrages) while preserving reliable 30s taste-gate pre-seed on every START/RETRY.
- Sharp feedback: immediate particles on input, gate shatter arcs+burst at exact crossing y (stronger on beat), floating LANE/POLARITY miss toasts, top-of-lane "!" glitch warnings (250-350ms telegraph), HUD, screen shake/flash on death, combo decay when idle.
- Controls: kb (arrows/A-D + SPACE), full-height pointer/touch strips (any Y on left/center/right thirds), buttons. All ≥44px targets. Responsive 400x700 canvas in aspect-ratio container (mobile full-bleed).
- Rave-bright TB-123 house style: neon #00FF41/#FF006E/#3A86FF on dark, CRT scanlines, reactive bg wash to current polarity, interference visuals on glitches (tears + fringing).
- Self-contained, <32kB source, zero external requests, audio only after gesture (WebAudio tones), 60fps trivial, offline post-load.
- Restart, score/max-combo on gameover, pre-seeded curated first slice so new player can act in <5s and evaluate the dual-match verb without RNG wait.
- All prior polish (shatter, toasts, warnings, beat-phase, full touch, patterns) retained; no scope creep.
