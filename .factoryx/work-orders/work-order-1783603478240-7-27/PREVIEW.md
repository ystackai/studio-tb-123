# Preview — Pocket Signal Machine

## Preview Path
`games/pocket-signal-machine/index.html`

## How to Play
1. Click **POWER ON** or press **Space** to start
2. Tap the **8 pads** (Q W E R / A S D F) to trigger drum and synth sounds
3. Adjust **FILTER** (arrow keys) to sweep the sound from muffled to bright
4. Adjust **DELAY** (Z / X keys) to add echo
5. Hit **8+ pads in a row** to trigger the **CHAIN payoff** — the machine lights up with expanding rings and the payoff sound

## Visual Elements
- 8 color-coded pads in a 2×4 grid
- 2 rotary knobs (FILTER, DELAY) with draggable arcs
- Signal-flow cables with animated pulses from pads → filter → delay → output
- Output meter that responds to signal activity
- Particle bursts on each pad hit
- Waveform image decoration from foundry assets
- Chain counter and payoff flash effect

## Assets
- 1 music loop (foundry_music_loop.wav, 5.3MB, 31s)
- 8 SFX samples from foundry cozy_audio_pack
- 2 waveform PNG visual decorations from foundry

## Payload
- Total: ~6MB (mostly audio)
